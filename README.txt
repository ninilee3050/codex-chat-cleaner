Codex 채팅 삭제기

실행:
1. "Run Codex Chat Cleaner.bat" 파일을 더블클릭합니다.
2. 삭제할 채팅의 체크박스를 누릅니다.
3. "체크한 항목 삭제" 버튼을 누릅니다.
4. 생성 이미지는 상단의 "이미지 관리" 버튼을 눌러 같은 화면에서 관리합니다.

파일 역할:
- Run Codex Chat Cleaner.bat: 더블클릭 실행용 파일입니다. 가능하면 CMD 창 없이 실행합니다.
- codex_chat_cleaner.py: 실제 프로그램 본체입니다. 삭제하면 안 됩니다.
- README.txt: 이 설명 파일입니다.

목록:
- 일반 채팅 세션만 표시하고 검색합니다.
- "The following is..."로 시작하는 내부 검토 기록은 목록에서 숨깁니다.
- 출처가 guardian으로 표시되는 내부 검토 기록도 목록에서 숨깁니다.
- 수정일 기준 최신순으로 정렬됩니다.
- 출처/모델 열은 참고 정보입니다.
- "열기" 버튼은 해당 채팅 세션이 사용한 작업 폴더를 엽니다.

버튼:
- 새로고침: Codex 채팅 목록을 다시 읽습니다.
- 이미지 관리: 채팅 목록 영역을 생성 이미지 미리보기 목록으로 바꿉니다.
- 보이는 항목 모두 체크: 현재 검색 결과에 보이는 항목을 전부 체크합니다.
- 체크 해제: 모든 체크를 풉니다.
- 체크한 항목 삭제: 체크한 채팅을 삭제합니다.

삭제 동작:
- Codex 최근 목록 DB에서 해당 채팅을 제거합니다.
- 실제 세션 .jsonl 파일도 함께 삭제합니다.
- 해당 채팅 ID를 참조하는 내부 검토 기록도 자동으로 함께 삭제합니다.
- 해당 채팅이 쓰던 작업 폴더가 비어 있으면 함께 삭제합니다.
- 작업 폴더 안에 파일이 있으면 자동 삭제하지 않습니다.
- 관련 인덱스/전역 상태 항목이 있으면 같이 정리합니다.

이미지 관리:
- C:\Users\user\.codex\generated_images 안의 이미지 파일을 최신순으로 보여줍니다.
- 이미지 미리보기를 갤러리 형태로 보고 체크한 뒤 "선택 이미지 삭제"로 지울 수 있습니다.
- 갤러리는 최소 4칸으로 보이고, 프로그램 창을 넓히면 칸 수가 자동으로 늘어납니다.
- 이미지가 많으면 같은 목록 안에서 스크롤됩니다.
- "채팅 세션"과 "이미지 관리" 버튼은 항상 같은 위치에 고정됩니다.
- 왼쪽 통계 문구도 현재 화면에 맞춰 "전체 세션" 또는 "전체 이미지"로 바뀝니다.
- 선택한 이미지가 삭제되어 비어버린 하위 폴더는 함께 삭제합니다.
- generated_images 폴더 바깥의 파일은 삭제하지 않습니다.

주의:
- 중요한 채팅은 먼저 Codex의 보관 기능을 사용하세요.
- 삭제기에는 별도 백업 기능이 없습니다.
- 사용 중인 Codex 창이 목록을 오래 들고 있을 수 있으니, 삭제 후 Codex를 다시 열면 더 깔끔합니다.
